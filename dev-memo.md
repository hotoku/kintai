## 時間の扱い

``` typescript
const x = new Date(1970, 0, 1, 0, 0, 0)
conosle.log(x)
// => 1969-12-31T15:00:00.000Z
```

- `new Date`では、ローカルのタイムゾーンでの時刻と解釈される
- 表示の際には、タイムゾーン付でUTCで表示される
